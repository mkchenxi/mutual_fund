Description

This script describes how the bank- and fund-side networks evolve over time. The focus is on the possible "spillover" between banks and funds. I adopt the following steps to obtain and analyze the bank networks and fund firm ("firm" for short) networks.
  1. Tag data by months. In total, there are 5 years of data or 60 months.
  2. For each month, build the bank-firm two-mode network. If a bank is ever matched to a firm in a month, the entry is 1. Otherwise, it's 0.
  3. Bank and firm networks are constructed from the bank-firm two-mode networks. If two banks (firms) are matched to the same firms (banks), then the entry is 0. Otherwise, it's 1.
  4. From Step 3, we have 60 (months) of bank and firm networks. To see the possible "spillover" between banks or firms, I check the similarities between the time t network and time network. Due to the sparsity of the networks, I remove entries that are both zeros in both the time t network and the time network. I use a similarity measure: , or out of all the connected pairs in both newtork tand network , how many pairs are always connected. The larger this value, the more similar the two networks.

Data Preparation

Data loading and rendering

% This script prepare the network
clear;
load("matlab_data.mat","fund","match");
fund.Properties.VariableNames{2} = 'Fund_ID';
% remove banks with id's [5,7,9,13,16,33,39]
match = match(~ismember(match.Bank_ID,[5,7,9,13,16,33,39]),:);
% relabel banks
match.Bank_ID = sum(bsxfun(@times,bsxfun(@eq, match.Bank_ID, ...
unique(match.Bank_ID)'),1:42),2);
% designate the time of matches
match = join(match, fund(:, [2,17,20]), "Keys", "Fund_ID");

Creating dynamic two-mode networks

% use a loop to create networks one by one
nb = max(match.Bank_ID);
nf = max(match.Firm_ID);
nt = max(match.Month_ID);
% to use months as the time periods
two_mode = cell(nt, 1);
% the binary fund-bank networks from month 1 to 60
for i = 1:nt
match_i = unique(match(match.Month_ID==i, 2:3), 'rows');
two_mode{i} = sparse(match_i.Firm_ID, match_i.Bank_ID, ...
ones(size(match_i,1),1), nf, nb);
end

Creating and examining the bank and fund firm networks

The bank networks

% create the bank network
bank_net = two_mode;
bank_net = cellfun(@(x) x'*x, bank_net, 'UniformOutput',false);
% convert to binary networks and set the self-links to zeros
bank_net = cellfun(@(x) (x>0).*~eye(nb), bank_net, 'UniformOutput',false);
% checking the similarities between t-1 and t network
bank_net_corr = zeros(nt-1,1);
for i = 2:nt
net1 = bank_net{i-1}(:);
net2 = bank_net{i}(:);
idx = net1 + net2;
bank_net_corr(i-1) = sum(idx==2)/sum(idx>0);
end
The plot below shows the similarities between bank networks. It seems there are cases where the values are relatively low, and also cases where they are high. The average value is the similarity is 0.52. A rough interpreation is in two consecutive months, 52% banks stay with the same fund firms, and 48% with different ones. The plot of the data is below.
% Create plot of bank_net_corr
plot(bank_net_corr);
% Add ylabel, title, and legend
legend('off')
ylabel('Bank Network Similarities')
title('Bank Network Similarities (0s removed)')
yline(mean(bank_net_corr),'-','Mean Value: 0.52');

The firm networks

The next step is to apply the same procedure to get the firm networks.
